## Objetivo do curso

- Praticar conceitos iniciais de Numpy e operações com arrays.  
- Aprender indexação, fatiamento, funções universais, reshape e manipulação avançada de arrays.  
- Aplicar o conhecimento em mini-projetos para fixar a prática.  
- Familiarizar-se com **Jupyter Notebooks** e **Git** para controle de versão e portfólio.

---

**Seção 10 – Consolidando conhecimentos**

## Como usar

1. Clone o repositório:

```bash
git clone https://github.com/brunodutraho/curso-numpy.git
````
2. Entre na pasta do projeto:
```bash
cd curso-numpy
````
3. Ative o ambiente virtual:
```bash
.\.venv\Scripts\Activate.ps1  # PowerShell
# ou
.\.venv\Scripts\activate      # CMD
````
4. Instale as dependências:
```bash
pip install -r requirements.txt
````
5. Abra o Jupyter Notebook:
```bash
jupyter notebook
````
6. Navegue até a pasta notebooks/ e abra os arquivos .ipynb.
